### F13 AI换模特

知识块类型：功能说明
功能编号：F13
功能名称：AI换模特
相关标签：AI换模特，POP AI智绘，AI设计，时尚AIGC
功能定位：替换图片中的模特人物，支持性别、风格、场景需求。
适用场景：电商上新、不同人群展示、多模特素材生成。
输入要求：上传清晰模特图；目标模特风格明确；
输出结果：更换模特后的展示图。

操作流程摘要：
1. 上传身穿服装的模特图。
2. 选择目标模特。
3. 生成替换结果。
4. 预览并保存。

使用建议：
- 整头替换或跨性别替换可能影响自然度，需人工复核。

常见问题摘要：
- 问题：换模特后服装变了。处理：推荐使用F01 AI改款 ，提示增加保留原服装、姿势、光影和背景。

场景案例：
场景：多模特展示
案例：将同一服装替换到不同模特形象上。
价值：减少重新拍摄成本。

是否需要转人工：一般操作问题不需要；连续多次生成失败、涉及正式报价/合同/版权法律判断、企业私有化/API/权限开通、客户投诉或高价值商机时，建议转人工。
