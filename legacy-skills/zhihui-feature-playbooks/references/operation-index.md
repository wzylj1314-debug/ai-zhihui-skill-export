# 06 操作流程说明

AI阅读说明：本章为具体操作步骤。

### F01 AI改款

1. 上传原款式图。
2. 选择 AI改款。
3. 输入修改需求，明确变化内容与保留内容。
4. 生成并预览结果。
5. 下载或继续微调。

### F02 一键改款

1. 上传款式图。
2. 设置原图相似度。
3. 点击生成多版结果。
4. 预览并筛选可用方案。

### F03 款式融合

1. 上传款式图 A。
2. 上传款式图 B。
3. 调整原图相似度。
4. 预览并保存。

### F04 相似款衍生

1. 选择风格引擎。
2. 上传参考款式图。
3. 调整合理选区。
4. 调整相似度并补充或修改关键词。
5. 预览并保存。

### F05 百变款式

1. 上传原款式图。
2. 上传参考款。
3. 调整风格强度。
4. 补充标签/关键词（非提示词模板）。
5. 预览并保存。

### F06 文生款

1. 选择风格引擎。
2. 输入款式描述。
3. 选择出图方向：款式图或模特图。
4. 选择生成数量。
5. 生成、预览并下载。

### F07 线稿生款

1. 上传线稿图。
2. 上传参考款式图。
3. 选择单品标签。
4. 补充标签/关键词（非提示词模板）。
5. 预览并保存。

### F08 款生线稿

1. 上传款式图。
2. 生成线稿。
3. 预览并保存。

### F09 虚拟试衣

1. 上传款式图。
2. 选择模特或上传自定义模特图。
3. 涂抹完整服装选区。
4. 生成试衣图。
5. 预览并保存。

### F10 视频试衣

1. 上传款式图和模特素材。
2. 选择或描述动作。
3. 生成视频。
4. 预览并保存。

### F11 参考生视频

1. 上传 1–4 张参考图。
2. 输入视频描述。
3. 生成视频。
4. 预览并保存。

### F12 图生视频

1. 上传模特图。
2. 选择动作模板或输入自定义动作。
3. 生成视频。
4. 预览并保存。

### F13 AI换模特

1. 上传身穿服装的模特图。
2. 选择目标模特。
3. 生成替换结果。
4. 预览并保存。

### F14 AI换背景

1. 上传图片。
2. 选择背景方式：推荐场景或自定义。
3. 预览并保存。

### F15 AI褪底

1. 上传图片。
2. 选择去背景或替换背景色。
3. 预览抠图结果。
4. 下载或进入后续编辑。

### F16 涂抹消除

1. 上传图片。
2. 涂抹需要去除或修复的区域。
3. 生成修复结果。
4. 预览并下载。

### F17 AI拆款

1. 上传拆款图片。
2. 涂抹需要拆款的服装区域。
3. 生成平铺图。
4. 预览并保存。

### F18 AI描稿

1. 上传图案或款式图。
2. 框选花型区域。
3. 普通高清描稿可按实际需求选择模式；如果做四方连续，选择精准模式。
4. 如果做四方连续，选择四方连续。
5. 预览并保存。

### F19 AI转矢量

1. 上传花型或线稿图。
2. 选择精调版/极速版。
3. 预览矢量化效果。
4. 下载 SVG/EPS。

### F20 以图生文

1. 上传图片。
2. 查看生成描述或关键词。
3. 编辑或复制用于文生款/文生图。

### F21 文生图

1. 输入图案描述。
2. 使用自动扩写。
3. 选择风格引擎和标签。
4. 选择尺寸。
5. 生成并保存。

### F22 相似图衍生

1. 选择风格引擎。
2. 上传图案。
3. 调整相似度或创新幅度。
4. 输入关键词。
5. 预览并保存。

### F23 百变花型

1. 上传基础图案。
2. 上传风格图。
3. 调整风格强度和生成方向。
4. 生成新花型。
5. 预览并保存。

### F24 图案融合

1. 上传图案 A。
2. 上传图案 B。
3. 调整相似度和标签。
4. 生成融合图案。
5. 预览并保存。

### F25 图案配色

1. 上传图案。
2. 选择推荐配色、参考配色或定向换色。
3. 调整参数。
4. 预览并保存。

### F26 款式配色

1. 选择款式配色。
2. 上传原款式图。
3. 输入配色需求，说明需要修改的颜色区域与目标色彩方向。
4. 生成并预览结果。
5. 下载结果或继续调整配色方案。

### F27 花型上身

1. 选择花型上身。
2. 上传服装款式图。
3. 上传或指定需要应用的花型图。
4. 选择花型密度和透明度。
5. 生成并预览结果。
6. 下载结果。

### T01 高清放大

1. 选择 高清放大。
2. 上传需要处理的图片。
3. 选择或确认放大需求。
4. 生成高清结果并预览。
5. 下载高清图片或继续用于其他功能处理。
