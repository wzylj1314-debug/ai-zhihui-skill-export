### F21 文生图

知识块类型：功能说明
功能编号：F21
功能名称：文生图
相关标签：文生图，POP AI智绘，AI设计，时尚AIGC
功能定位：将文字描述直接转化为创意图案或花型。
适用场景：花型创意、印花探索、图案灵感提案。
输入要求：按“元素 + 配色 + 主体 + 风格 ”描述；可使用自动扩写。
输出结果：原创图案或花型图。

操作流程摘要：
1. 输入图案描述。
2. 使用自动扩写。
3. 选择风格引擎和标签。
4. 选择尺寸。
5. 生成并保存。

使用建议：
- 描述应包含元素、构图、配色、风格和用途；如果需要四方连续、连续花型或满印循环，应改用 AI描稿。
- 原创花型建议多生成几版，再用 F22 或 F25 做系列延展。
- 涉及品牌图案、IP形象或受保护元素时，应提醒用户避免侵权。

常见问题摘要：
- 问题：花型不可满印。处理：文生图不用于四方连续/满印循环；如需要四方连续或连续花型，应改用 AI描稿，选择精准模式并选择四方连续。

输入描述参考：
- 浅粉色底，点缀小花朵，低饱和莫兰迪色系，清新春夏女装印花。
- 几何元素，渐变色，轻奢风格，适合丝巾和女装面料图案。

场景案例：
场景：原创花型探索
案例：输入花型描述生成原创图案。
价值：快速获得灵感方案。

是否需要转人工：一般操作问题不需要；连续多次生成失败、涉及正式报价/合同/版权法律判断、企业私有化/API/权限开通、客户投诉或高价值商机时，建议转人工。
