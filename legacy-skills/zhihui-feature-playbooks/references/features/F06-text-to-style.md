### F06 文生款

知识块类型：功能说明
功能编号：F06
功能名称：文生款
相关标签：文生款，POP AI智绘，AI设计，时尚AIGC
功能定位：根据文字描述生成款式图或模特图，系统可扩写风格、剪裁、材质、色彩等信息。
适用场景：概念设计、提案图、款式初稿。
输入要求：描述包含单品、版型、材质、颜色、风格、适用场景；可使用自动扩写。
输出结果：款式图或模特图，可标清或超清输出。

操作流程摘要：
1. 选择风格引擎。
2. 输入款式描述。
3. 选择出图方向：款式图或模特图。
4. 选择生成数量。
5. 生成、预览并下载。

使用建议：
- 文字描述应包含单品、版型、材质、颜色、风格、季节和使用场景。
- 用于客户提案时建议生成多版，不要把第一版结果直接当最终款。
- 如果结果太泛，补充品牌调性、细节限制。

常见问题摘要：
- 问题：生成款式太普通。处理：补充风格、目标人群、面料、细节和场景。

输入描述参考：
- 生成一件夏季连衣裙，奶油白配色，泡泡袖，低饱和花卉图案，高级通勤风。
- 宽松针织毛衣，低饱和灰蓝色，圆领，简约设计，柔软羊毛质感。

场景案例：
场景：概念设计落地
案例：输入“夏季连衣裙，奶油白，泡泡袖，高级通勤风”生成款式图。
价值：无需手绘即可形成初稿。

是否需要转人工：一般操作问题不需要；连续多次生成失败、涉及正式报价/合同/版权法律判断、企业私有化/API/权限开通、客户投诉或高价值商机时，建议转人工。
