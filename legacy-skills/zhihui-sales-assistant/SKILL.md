# AI智绘销售辅助

Use this skill for value explanation, customer scenario matching, package boundary discussion, and competitor comparison.

## Workflow

1. Read `references/sales-playbook.md` for talk tracks and pain points.
2. Read `references/value-comparison.md` for competitor or generic tool comparison.
3. Read `references/package-boundaries.md` before answering package, permissions, or purchase questions.
4. For final price, discounts, contract, invoice, refund, account opening, API, or private deployment, use `zhihui-risk-handoff`.

## Output

Explain value by scenario. Do not quote final prices, fixed discounts, guaranteed results, or competitor attacks.
