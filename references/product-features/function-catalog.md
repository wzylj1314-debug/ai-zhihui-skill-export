# 03 功能编号总览

## 3.1 主功能统一编号

- 1：F01
  - 编号：F01
  - 功能名称：AI改款
  - 功能定位：在原款基础上进行局部或整体修改，支持颜色、材质、结构、图案、元素增减等方向。
  - 典型场景：服装、鞋包、家居、首饰等已有款式的快速改款和系列延展。
- 2：F02
  - 编号：F02
  - 功能名称：一键改款
  - 功能定位：基于原款快速生成多种创新变体，系统自动调整局部结构、配色和材质纹理。
  - 典型场景：已有爆款/基础款需要快速生成多版方案。
- 3：F03
  - 编号：F03
  - 功能名称：款式融合
  - 功能定位：上传两张不同款式图，提取两者设计元素并生成融合款式。
  - 典型场景：跨风格组合、系列创新、参考款混搭。
- 4：F04
  - 编号：F04
  - 功能名称：相似款衍生
  - 功能定位：基于参考款生成多款相似但有变化的设计，可做全局或局部衍生。
  - 典型场景：热销款延展、系列开发、局部改款。
- 5：F05
  - 编号：F05
  - 功能名称：百变款式
  - 功能定位：上传原款和参考款，保留廓形并迁移配色、材质、细节，生成多款变体。
  - 典型场景：同版多变体、风格迁移、系列开发。
- 6：F06
  - 编号：F06
  - 功能名称：文生款
  - 功能定位：根据文字描述生成款式图或模特图，系统可扩写风格、剪裁、材质、色彩等信息。
  - 典型场景：概念设计、提案图、款式初稿。
- 7：F07
  - 编号：F07
  - 功能名称：线稿生款
  - 功能定位：根据黑白线稿生成完整服装款式图，自动补全配色、纹理和面料效果。
  - 典型场景：线稿视觉化、款式效果图生成、结构方案上色。
- 8：F08
  - 编号：F08
  - 功能名称：款生线稿
  - 功能定位：将款式图生成黑白线稿，用于二次设计、结构提取或手绘参考。
  - 典型场景：款式图转设计线稿、二次修改、版型参考。
- 9：F09
  - 编号：F09
  - 功能名称：虚拟试衣
  - 功能定位：将款式图映射到模特或场景中，生成真实感上身效果。
  - 典型场景：电商主图、营销展示、设计提案、快速试穿演示。
- 10：F10
  - 编号：F10
  - 功能名称：视频试衣
  - 功能定位：基于 AI 人模合成生成试穿动态视频。
  - 典型场景：短视频营销、动态选款、视觉提案。
- 11：F11
  - 编号：F11
  - 功能名称：参考生视频
  - 功能定位：通过多张参考图和文字描述生成创意服装展示视频。
  - 典型场景：社交媒体宣传、电商视频、搭配展示、视觉提案。
- 12：F12
  - 编号：F12
  - 功能名称：图生视频
  - 功能定位：将款式图结合动作模板或自定义动作生成拟真人穿搭视频。
  - 典型场景：电商营销、产品展示、社媒推广。
- 13：F13
  - 编号：F13
  - 功能名称：AI换模特
  - 功能定位：替换图片中的模特人物，支持性别、风格、姿态或场景需求。
  - 典型场景：电商上新、不同人群展示、多模特素材生成。
- 14：F14
  - 编号：F14
  - 功能名称：AI换背景
  - 功能定位：快速替换模特或产品图背景，支持预设背景或自定义场景。
  - 典型场景：电商展示、广告制作、产品视觉统一。
- 15：F15
  - 编号：F15
  - 功能名称：AI褪底
  - 功能定位：一键去除复杂背景，提取主体并保留轮廓。
  - 典型场景：款式图、模特图、商品图抠图，便于后续换背景或排版。
- 16：F16
  - 编号：F16
  - 功能名称：涂抹消除
  - 功能定位：通过涂抹去除图片中不需要的元素，系统自动补全背景。
  - 典型场景：局部修整、杂物去除、瑕疵处理、图片修复。
- 17：F17
  - 编号：F17
  - 功能名称：AI拆款
  - 功能定位：上传秀场 Look 或商拍图，识别版型结构与设计细节，生成平铺图或初步设计稿。
  - 典型场景：快时尚拆款、趋势响应、秀场款快速分析。
- 18：F18
  - 编号：F18
  - 功能名称：AI描稿
  - 功能定位：将模糊款式图、实物印花或图案转为高清图，支持连续花型生成。
  - 典型场景：服装设计、印刷制图、花型提取、图案修复。
- 19：F19
  - 编号：F19
  - 功能名称：AI转矢量
  - 功能定位：将花型或线稿图生成高质量矢量图，边缘锐利，可按颜色分层。
  - 典型场景：印花、刺绣、激光切割、专业制图。
- 20：F20
  - 编号：F20
  - 功能名称：以图生文
  - 功能定位：解析参考图生成高质量关键词，可直接用于文生款或文生图。
  - 典型场景：快速生成提示词、关键词补全、提示词优化。
- 21：F21
  - 编号：F21
  - 功能名称：文生图
  - 功能定位：将文字描述直接转化为创意图案或花型。
  - 典型场景：花型创意、印花探索、图案灵感提案。
- 22：F22
  - 编号：F22
  - 功能名称：相似图衍生
  - 功能定位：基于原图特征生成风格相似但有变化的多样化图案。
  - 典型场景：花型延展、系列图案开发、创意扩散。
- 23：F23
  - 编号：F23
  - 功能名称：百变花型
  - 功能定位：在基础图案上结合参考原图和风格图，生成全新花型。
  - 典型场景：花型风格迁移、图案创意延展。
- 24：F24
  - 编号：F24
  - 功能名称：图案融合
  - 功能定位：提取两张图案的色彩、纹理、构图，生成融合新图案。
  - 典型场景：花型差异化、图案创新、系列开发。
- 25：F25
  - 编号：F25
  - 功能名称：图案配色
  - 功能定位：支持推荐配色、参考配色和定向换色，对原图进行配色方案生成。
  - 典型场景：花型改色、系列色板开发、趋势色验证、局部换色。
- 26：F26
  - 编号：F26
  - 功能名称：款式配色
  - 功能定位：支持参考配色和精准改色，对原款式进行配色方案生成。
  - 典型场景：系列色板开发、趋势色验证、局部换色。
 - 27：F27
  - 编号：F27
  - 功能名称：花型上身
  - 功能定位：将花型图映射到服装区域，用于查看印花上身效果。
  - 典型场景：花型、图案、印花上身效果查看，辅助开发。

## 3.2 扩展/辅助能力保留说明

- 1：T01
  - 编号：T01
  - 能力名称：高清放大
  - 归属说明：作为辅助优化工具，对款式图、模特图、图案图进行 2×/4× 放大与细节增强。

## 3.3 功能归类

- 1：款式改款与延展
  - 类型：款式改款与延展
  - 对应功能：F01 AI改款、F02 一键改款、F03 款式融合、F04 相似款衍生、F05 百变款式、F26款式配色
- 2：文字/线稿生成款式
  - 类型：文字/线稿生成款式
  - 对应功能：F06 文生款、F07 线稿生款、F08 款生线稿、F01 AI改款
- 3：试衣与视频
  - 类型：试衣与视频
  - 对应功能：F09 虚拟试衣、F10 视频试衣、F11 参考生视频、F12 图生视频、F01 AI改款
- 4：电商图像处理
  - 类型：电商图像处理
  - 对应功能：F13 AI换模特、F14 AI换背景、F15 AI褪底、F16 涂抹消除、F01 AI改款
- 5：拆款与制图
  - 类型：拆款与制图
  - 对应功能：F17 AI拆款、F18 AI描稿、F19 AI转矢量、F20 以图生文
- 6：图案创作与延展
  - 类型：图案创作与延展
  - 对应功能：F21 文生图、F22 相似图衍生、F23 百变花型、F24 图案融合、F25 图案配色、F27花型上身、F01 AI改款
