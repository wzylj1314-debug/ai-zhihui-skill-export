# 07 AI改款提示词参考

本文件用于 `AI改款` 的提示词写法教学与参考。目标是教用户学会自己写，而不是直接甩成品模板（授人以渔）。

适用范围：本文件只用于 `AI改款`。不要把它用于 `文生款`、`文生图`、`相似款衍生`、`相似图衍生`、`AI描稿`、`百变花型`、`AI转矢量`、`虚拟试衣`、`去背景`、`涂抹消除` 等功能。

四方连续、连续花型、满印循环、边缘衔接、无缝拼接只能用 `AI描稿`，且 `AI描稿` 没有提示词输入；回答这类问题时不要给提示词参考、模板或示例。只回答：上传清晰图案、框选准确、选择精准模式、选择四方连续、生成后检查边缘拼接；不推荐 `百变花型`，不说智能模式，不说满印方式。

只有当用户问题同时指向 `AI改款`，且在问"提示词怎么写 / 怎么描述 / 效果不好怎么调 / 怎么只改局部"时，才进入本教学口径；入口、步骤、价格、权限、报错、功能区别等问题继续按原有规则回答。

## 0. 回答总原则（温和诊断，再教方法，最后示范）

用户问 `AI改款` "提示词怎么写 / 怎么描述 / 效果不好怎么调 / 怎么只改局部"时：

1. 先温和诊断缺什么：保留项、修改部位、目标效果，还是描述太抽象。
2. 给可复用写法，不要直接甩成品模板。
3. 用用户自己的需求改写出一条示范提示词，并点出补上的保留项/具体特征。

语气要温和指导，不说"你写错了""这样不行""你没写清楚"。钉钉回复 ≤5 行，先结论。

## 1. 核心公式：保持 X，把 Y 改为 Z

AI改款更稳的写法是先说清楚"保留什么"，再说"改什么"：

`保持[保留项]，把[修改部位/维度]改为[目标效果]`

保留项可以包括：

- 版型、廓形、整体比例、衣长
- 领口、袖型、门襟、口袋、结构线
- 图案位置、底色、配色关系
- 面料质感、垂感、光泽、纹理

示范：

- 保持原来的宽松版型、衣长和门襟结构，只把袖子改为泡泡袖。
- 保持原版型和图案位置，把主色改为低饱和大地色系。
- 保持哑光垂软的面料质感，把颜色改为深色系。

## 2. 参考款写法：先找形 / 色 / 质打动点

用户有参考款，想保留一点、改一点时，先温和引导找打动点：

- 形：版型、廓形、比例、结构、肩线、腰线、衣长。
- 色：配色、色调、图案、色彩关系、深浅关系。
- 质：面料、质感、垂感、肌理、光泽、厚薄。

再用公式组装：`保持[打动点描述]，把[想变的部分]改为[变化方向]`。

示范：

- 打动点是形、想换色：保持宽松落肩廓形，把配色改为低饱和大地色系。
- 打动点是色、想换廓形：保持低饱和印花图案，把版型改为宽松 A 字剪裁。
- 打动点是质、想换色：保持哑光垂软的面料感，把颜色改为深色系。

## 3. 常见问题怎么温和教

- 整件都变：通常需要补"保留什么"，如保留版型、衣长、颜色、图案位置和结构。
- 一次改太多：建议一轮只改一个核心点，满意后再继续改下一处。
- "图一/图二"指代不清：改用可识别名称，如"浅蓝色鞋子为主图，参考黑色皮鞋的厚底结构"。
- 抽象词太多：把"高级感/年轻一点/更时尚"换成颜色、材质、纹理、光泽、比例等具体特征。
- 局部 AI 认不出：先用外部工具在图上画圈、箭头或文字标注，再上传。

## 4. AI改款提示词结构（兜底公式）

主体/单品 + 修改动作 + 保留内容 + 色彩/材质 + 输出要求。

要点：

- 修改动作明确，例如"将袖子改为泡泡袖"。
- 保留内容明确，例如"保留原版型轮廓和衣身长度"。
- 一次只改一个核心点，避免同时要求太多变化。
- 部位容易被误解时，建议先用外部工具在图上画圈、箭头或文字标记，再上传。

## 5. AI改款提示词示范（仅作改写参考，不要整段甩给用户）

引用例句时，标题必须写成：`AI改款提示词参考（仅用于 AI改款）`，并尽量改写成贴合用户需求的版本。

局部改款：

- 只将袖子改为泡泡袖，保留衣身版型、衣长、颜色和门襟结构。
- 将领口改为 V 领，袖口增加轻微荷叶边，保留原衣身轮廓和整体比例。
- 去掉胸前大蝴蝶结，保留衣服版型和颜色，增加简洁褶皱细节。

颜色 / 材质改款：

- 将主色改为奶油白，材质更轻薄，保留原服装结构和细节比例。
- 保留原版型和图案位置，只把面料质感改为柔软针织，颜色改为浅灰蓝。
- 将外套材质改为哑光皮革，保留廓形、口袋、门襟和袖口结构。

图案改款：

- 将上衣图案改为小花点缀，配色柔和，保留底色和原衣服结构。
- 保留原图主体和服装版型，只调整图案风格和颜色层次，其他部分不变。
- 将图案风格改为低饱和法式复古，保留图案在服装上的位置和比例。

系列延展 / 款式融合 / 换模特换背景：

- 基于原款生成同风格系列款，保留基础版型，调整配色和局部细节。
- 保留 A 款廓形，融合 B 款的配色和材质质感，生成新的通勤风上衣。
- 将模特替换为年轻亚洲女性，保持原服装、姿势、光影和背景不变。
- 将背景替换为浅灰色摄影棚，光线柔和，保留模特和服装边缘自然。

鞋包/家居/首饰同理：把"衣身/版型"替换为对应部件，如鞋面、鞋底、包身、五金、柜体、椅背、链条、宝石等。

## 6. 禁止误用

- 用户问 `文生款`、`文生图`、`相似款衍生`、`相似图衍生`，不要按本文件做提示词教学。
- 用户问 `AI描稿`，不要给提示词。
- 用户问四方连续、连续花型、满印循环、边缘衔接、无缝拼接，不要给提示词，不推荐 `百变花型`，不引用本文件任何提示词。
- 不说 `AI改款` 有内置涂抹、画笔、局部模式、选区、强度档位或保守档。
